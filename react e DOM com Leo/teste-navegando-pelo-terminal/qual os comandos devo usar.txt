O comando para criar pasta no terminal é: "mkdir (nome da pasta)"
---------------

O comando para entrar em uma pasta pelo terminal é: "cd (nome da pasta)"
---------------

E o comando para sair de uma pasta pelo terminal é: " cd.. "
---------------

Para ver a versão do npm que ta no pc é: "npm -v"
---------------

Para ver a versão do node que ta no pc é: "node -v"
---------------

Para baixar a biblioteca que eu quero do npm ( que por
sinal é onde fica o React) é: "i (nome da biblioteca)"
---------------

No vídeo ele pede para baixar a biblioteca "create-react-app"
---------------

Para criar um projeto devemos escrever o nome da da biblioteca
que vamos executar + o nome do projeto Ex:
"create-react-app projeto1"

Obs: Ele vai fica instalado apenas nessa pasta que colocamos o comando. 
Más se eu quiser instalar no PC inteiro o comando é: "-g "nome da biblioteca"
---------------
